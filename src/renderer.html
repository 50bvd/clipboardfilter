Format du template

Un template contient une ou plusieurs categories avec leurs filtres :

{
  "name": "Mon Template",
  "description": "Description",
  "author": "Votre nom",
  "version": "1.0.0",
  "filters": [
    {
      "category": "Custom",
      "description": "Mon filtre",
      "pattern": "regex",
      "replacement": "REDACTED",
      "useRegex": true,
      "enabled": true
    }
  ]
}
